# 产品需求文档（PRD）极简版：文档终版确定器 Text Unifier

| 项目名称 | 文档终版确定器（Text Unifier） |
| :--- | :--- |
| **版本号** | V1.0 |
| **目标平台** | Windows / macOS 桌面端（Electron 或 Tauri 技术栈建议） |

## 1. 功能边界
### 做什么
- 支持用户拖入/选择多个 `.txt` 文件。
- 扫描并识别不同文件中 **100% 相同** 的文本段落。
- 提供可视化列表，高亮展示重复段落，支持用户**手动勾选**删除项。
- 右侧实时预览合并后的最终文档内容。
- 鼠标悬停/点击预览区段落时，显示该段落**来源于哪些原始文件**。
- 导出纯净的合并后 `.txt` 文件。

### 不做什么
- 不支持 `.docx` / `.pdf` / `.mobi` 等格式（V1.0 范围外）。
- 不支持**近似语义匹配**（如 90% 相似度）。仅做严格字符串匹配。
- 不保留任何字体、颜色、加粗等富文本格式。
- 导出文件不包含来源批注/脚注。

## 2. 核心业务逻辑
### 2.1 文本归一化预处理规则
```
function normalizeText(rawText):
    // 1. 统一换行符为 \n
    // 2. 移除首尾空白字符（trim）
    // 3. 将连续多个空白符（空格/制表符）替换为单个空格
    // 4. 过滤不可见控制字符（除换行外）
    // 5. 按行分割，过滤空行（仅含空白符的行视为空行）
    return 处理后的段落数组 [段落A, 段落B, ...]
```

### 2.2 去重合并主流程
1. 用户添加文件列表 Files = [F1, F2, ...]
2. 并行读取所有文件内容并执行 normalizeText
3. 建立数据结构：
   - 段落指纹库：Map<段落内容Hash, { 首次出现文件, 所有出现位置列表 }>
4. 遍历所有段落，识别重复项：
   - IF 段落Hash已存在 AND 段落内容字符串严格相等（二次校验）: 标记为重复段落，记录来源文件集合
5. 生成合并预览文档：
   - 按文件添加顺序，依次提取**未标记删除**的段落（每个段落内容仅保留一次）
6. 前端渲染：
   - 左侧：展示所有重复段落组，每组提供 Checkbox 供用户勾选“是否删除该重复项”
   - 右侧：根据勾选状态实时刷新预览视图
7. 导出：
   - 将预览视图中的纯文本内容拼接，以 \n\n 分隔段落，写入 .txt 文件

## 3. 验收标准
| 编号 | 验收项 | 预期结果 |
| :--- | :--- | :--- |
| AC-01 | 文件导入 | 支持 `.txt` 文件多选/拖拽；拒绝非 `.txt` 后缀文件并提示。 |
| AC-02 | 空格容错 | `"Hello  World"` 与 `"Hello World"` 识别为重复。 |
| AC-03 | 换行容错 | 段落内软换行与硬换行归一后识别为重复。 |
| AC-04 | 乱码/特殊字符 | 零宽空格、BOM 头被清除，不影响匹配。 |
| AC-05 | 重复项高亮 | 左侧列表展示重复组，点击组名，右侧预览区对应段落高亮。 |
| AC-06 | 来源溯源 | 预览区悬停段落，浮层显示 `来源：文件A.txt, 文件C.txt`。 |
| AC-07 | 手动勾选逻辑 | 勾选重复项后，右侧预览立即移除该段落；取消勾选恢复。 |
| AC-08 | 导出功能 | 导出文件内容与右侧预览完全一致，无格式，UTF-8 编码。 |

# 产品交互原型（高保真，适配 AI 开发）

## 整体布局结构
```text
+-----------------------------------------------------------------------+
|  [文档终版确定器]                                          [ _ ][ □ ][ X ] |
+-----------------------------------------------------------------------+
|                                                                       |
|   [ 拖拽或点击上传 .txt 文件 ]   (已添加文件: 3 个)                      |
|                                                                       |
+-------------------------------+---------------------------------------+
|  左侧：重复段落列表 (3组)      |  右侧：最终文档预览                     |
|                               |                                       |
|  [✓] 重复组1 (出现2次)         |  这是第一段独有的内容。                 |
|      原文: "项目启动会议纪要"    |                                       |
|      来源: A.txt, B.txt       |  项目启动会议纪要  [来源浮层: A.txt, B.txt]|
|  [ ] 重复组2 (出现3次)         |                                       |
|      原文: "待办事项：..."      |  待办事项：                            |
|      来源: A.txt, B.txt, C.txt|    1. 确认预算                         |
|                               |    2. 分配任务                         |
|  [✓] 重复组3 (出现2次)         |                                       |
|      原文: "附件见邮件"         |  以上内容来自多个文档的整合。            |
|      来源: B.txt, C.txt       |                                       |
|                               |                                       |
|  [ 应用更改 ]  (实时刷新，也可自动) |                                     |
+-------------------------------+---------------------------------------+
|  状态栏：就绪                                      [ 导出合并文档 (.txt) ] |
+-----------------------------------------------------------------------+
```

## 核心组件规格定义
| 组件名 | 类型 | 属性 / Props | 交互说明 |
| :--- | :--- | :--- | :--- |
| `FileDropZone` | 上传区 | `accept=".txt"`, `multiple` | 支持点击和拖拽，上传后触发文件解析。 |
| `DuplicateList` | 列表容器 | `items: DuplicateGroup[]` | 渲染重复段落组。 |
| `DuplicateItem` | 列表项 | `isChecked`, `onToggle`, `snippet`, `sources` | 点击 Checkbox 切换删除/保留状态。 |
| `PreviewPanel` | 预览区 | `paragraphs: PreviewParagraph[]` | 渲染合并后文本，段落间距 1.5 倍。 |
| `PreviewParagraph` | 预览段 | `text`, `sourceFiles[]`, `onHover` | 悬停时触发 `Tooltip` 展示来源文件列表。 |
| `Tooltip` | 浮层 | `visible`, `content` | 跟随鼠标，显示文件名列表。 |
| `ExportButton` | 导出按钮 | `onClick` | 将当前预览区文本拼接为 Blob 下载。 |

## 关键交互状态流转
```text

                                          |
                                          v

       |
       v (用户勾选/取消勾选)
[实时更新右侧预览视图 (Filter 掉勾选的重复组)]
       |

[生成 .txt Blob 下载]
