## 6. V2.0.2 新增风险项

| 风险 | 等级 | 说明 | 缓解措施 |
| :--- | :---: | :--- | :--- |
| `triggerReanalysis` 连续触发 | 🟡 低 | 快速拖拽时可能连续触发 scan_files | loading 状态互斥防止并发 |
| 动态导入 `scanFiles` | 🔵 低 | `import('../utils/ipc')` 增加异步路径 | 构建 bundle 分析显示正常 |
| Web Crypto API 兼容 | 🔵 低 | Tauri WebView2 均支持 | Edge/Chrome 内核原生支持 |
