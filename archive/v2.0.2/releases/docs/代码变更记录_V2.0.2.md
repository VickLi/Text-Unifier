## 文件级变更

### 1. `src/store/useStore.ts`

| Bug | 位置 | 变更说明 |
| :--- | :--- | :--- |
| BUG-024 | `formatDocumentAction` | `rebuiltCheckedMap` 改用 `paragraphId` 直接匹配，替代 `content_hash` 匹配 |
| BUG-025 | `revertFormatting` | 基于快照重建 Map，完全丢弃 `fmt_*` 残留 |
| BUG-026 | 新增 `triggerReanalysis` | ~30 行新 action，读取排序后路径调用 scanFiles，保留勾选状态 |
| BUG-028 | `computeContentHash` | 从自定义 DJB2 哈希改为 Web Crypto API `crypto.subtle.digest('SHA-256')` |

**API 变更**: `computeContentHash` 从同步函数变为 `async` 函数，调用处增加 `await`。

### 2. `src/components/FileSortList.tsx`

| Bug | 位置 | 变更说明 |
| :--- | :--- | :--- |
| BUG-026 | `handleDragEnd` | `reorderFiles` 后调用 `triggerReanalysis()` |

### 3. `src/components/FormatButton.tsx`

| Bug | 位置 | 变更说明 |
| :--- | :--- | :--- |
| BUG-029 | 导入区 + `excludedCount` | 移除 `computeExcludedCount` 导入，改为内联 filter 计算 |

### 4. `src-tauri/src/document_formatter.rs`

| Bug | 位置 | 变更说明 |
| :--- | :--- | :--- |
| BUG-027 | `is_protected_block()` | 中文逗号检测从 `any()` 改为比例阈值 `> 0.5` |
| BUG-030 | `list_marker_re` 正则 | 列表标记后 `\s` → `\s?`（可选空格） |
