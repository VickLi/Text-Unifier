## 四、性能优化记录

| 优化项 | 优化前 | 优化后 | 提升 |
| :--- | :--- | :--- | :--- |
| Regex 编译 | 每次 format() 新建 | `new()` 一次性编译 | ~10x |
| 诗歌检测阈值 | avg_len < 20, no_period > 0.7 | avg_len < 12, no_period > 0.85 | 误判率降低 |
| React.memo | 部分组件 | 全部展示组件 | 减少重渲染 |
| Zustand selector | 全量订阅 | 细粒度 selector | 每组件只订阅需要字段 |
| Map vs Object | `Set` 存储 | `Map<string, boolean>` | O(1) 查找 |
| Rust LTO | 默认 | `lto=true, codegen-units=1` | 二进制优化 |
