## 4. 风险评估

### 4.1 高风险区域

| 风险区域 | 风险等级 | 说明 |
| :--- | :---: | :--- |
| **排版后勾选状态一致性** | 🔴 **高** | BUG-024 会导致用户勾选操作在排版后丢失 |
| **排序后分析不触发** | 🟠 **中** | RQ-01 核心功能不完整，拖拽无实际效果 |
| **段落哈希不一致** | 🟠 **中** | 前端/后端哈希不兼容，可能影响未来功能 |
| **诗歌保护误判** | 🟡 **低** | 含标点诗歌被错误合并 |

### 4.2 风险缓解建议

| 风险 | 缓解措施 |
| :--- | :--- |
| 勾选状态丢失 | **P0 优先修复** BUG-024，修复后增加排版后勾选状态验证测试 |
| 排序不触发分析 | 实现拖拽后自动调用 `scan_files` 的逻辑 |
| 哈希不一致 | 统一使用 Rust 后端 SHA256 或浏览器原生 SubtleCrypto |
