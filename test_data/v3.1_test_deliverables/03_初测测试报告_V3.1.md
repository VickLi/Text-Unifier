## 1. 版本概述

V3.1 在 V3.0 (Electron + napi-rs) 基础上集成 novel-processor 的**小说文本清洗引擎**，新增 **5 大功能模块**：

| 需求 | 功能 | 实现方式 | 代码位置 |
| :--- | :--- | :--- | :--- |
| **RQ-04** | 繁简转换 | js-opencc（纯 JS 异步） | `novelProcessor.ts` |
| **RQ-05** | 章节识别/格式化/分割 | 正则匹配 + 字符串操作 | `novelProcessor.ts` |
| **RQ-06** | 章节重排 | 序号提取 + 排序 | `novelProcessor.ts` |
| **RQ-07** | 垃圾内容过滤 | 7 条正则替换 | `novelProcessor.ts` |
| **RQ-08** | 内容筛选 | 关键词匹配 + 长度豁免 | `novelProcessor.ts` |

**Rust 核心层变更最小化**：仅新增 2 个 `#[napi]` 函数（`detect_encoding`、`scan_preprocessed_texts`），原有 5 个 Rust 模块零改动。
