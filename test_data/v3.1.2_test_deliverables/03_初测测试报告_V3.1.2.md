## 1. 版本概述

V3.1.2 为小版本维护发布，聚焦**代码规范校验 + 兼容性修复**，无新增功能：

| 变更 | 类型 | 影响范围 |
| :--- | :---: | :--- |
| `ExportResult.saved_path` → `savedPath` | 🐛 Bug 修复 | 导出功能运行时兼容 |
| `setAnalysisResult` 参数 camelCase | 🐛 Bug 修复 | Store 类型安全 |
| `require()` → `await import()` ESM | 🐛 Bug 修复 | Vite ESM 模式兼容性 |
| `_updateChapterList` 统一管理 | ⚡ 优化 | 代码可维护性 |
| Rust `#[allow(dead_code)]` warning | 🧹 清理 | 编译整洁度 |
