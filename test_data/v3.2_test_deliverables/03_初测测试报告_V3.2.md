## 1. 版本概述

V3.2 聚焦三大方向，**Rust 引擎零改动**，**IPC 零改动**：

| 需求 | 功能 | 变更量 | 实现 |
| :--- | :--- | :---: | :--- |
| RQ-01 | 文件芯片栏 | ~140 行 🆕 | FileChipBar + SortableChip |
| RQ-02 | 拖拽遮罩 | ~85 行 🆕 | DragOverlay + useGlobalDragDrop |
| RQ-03 | 编辑 + 5 步撤回 | ~70 行 🆕 + Store 扩展 | undoStack/pushSnapshot/undo/redo |
| RQ-04 | 预览主位 + 溯源 | PreviewPanel 增强 | ≥55% 宽度 + Tooltip 优化 |
| RQ-05 | 文档对比模式 | ~280 行 🆕 | DiffViewer + diffUtils(LCS+Levenshtein) |
