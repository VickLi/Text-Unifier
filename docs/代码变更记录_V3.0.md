## 一、新增文件（7 个）

### 1.1 `native/Cargo.toml`
napi-rs 项目配置。移除 Tauri/tauri-plugin 依赖，添加 `napi`、`napi-derive`、`napi-build`。crate-type 改为 `["cdylib"]`。

### 1.2 `native/build.rs`
```rust
extern crate napi_build;
fn main() { napi_build::setup(); }
```

### 1.3 `electron/main.ts`（~115 行）
Electron 主进程入口，注册 4 个 IPC handler：
- `scan-files` → napi `scanFiles()`
- `format-document` → napi `formatDocument()`
- `export-file` → `dialog.showSaveDialog` + `fs.writeFileSync`
- `select-files` → `dialog.showOpenDialog` + `fs.statSync`

包含：窗口管理（1200×800）、napi 模块加载、启动错误处理、生命周期管理。

### 1.4 `electron/preload.ts`（~12 行）
contextBridge 安全桥，暴露 4 个 API 到 `window.electronAPI`。

### 1.5 `electron/preload.d.ts`（~14 行）
`ElectronAPI` TypeScript 类型声明。

### 1.6 `electron-builder.yml`
Electron Builder 打包配置，NSIS 安装器，便携版。

### 1.7 `releases/v3.0/README.md`
V3.0 发布说明。

## 二、重写文件（2 个）

### 2.1 `native/src/lib.rs`（Rust napi 导出）

| 方面 | V2.0.2 | V3.0 |
| :--- | :--- | :--- |
| 属性宏 | `#[tauri::command]` | `#[napi]` |
| 错误类型 | `Result<T, String>` | `Result<T>`（napi::Error） |
| 函数注册 | `generate_handler![]` | 宏自动注册 |
| 并行收集 | `collect::<Result<Vec<_>, String>>()?` | `Vec<FileResult>` 显式错误收集 |
| 导出函数 | `scan_files`, `format_document`, `export_text` | `scan_files`, `format_document`（export_text 移至 Electron） |

### 2.2 `src/utils/ipc.ts`（前端通信层）

| 方面 | V2.0.2 | V3.0 |
| :--- | :--- | :--- |
| 底层调用 | `import { invoke } from '@tauri-apps/api/core'` | `getAPI().scanFiles(paths)` |
| 文件选择 | HTML `<input type="file">` | `electronAPI.selectFiles()` |
| 导出函数 | `exportText()` | `exportFile()` |
| 重试逻辑 | ✅ 保留 | ✅ 保留 |

## 三、微调文件（2 个）

### 3.1 `native/src/duplicate_resolver.rs`
- 添加 `#[napi(object)]` 宏到所有导出结构体
- `start_line: usize` → `u32`
- `occurrence_count: usize` → `u32`
- `file_size: u64` → `u32`
- `modified: u64` → `u32`
- `total_files: usize` → `u32`

### 3.2 `native/src/paragraph_index.rs`
- `start_line: (line_idx + 1) as u32`
- `occurrence_count: unique_file_count as u32`

### 3.3 `src/components/ExportButton.tsx`
- `import { exportText }` → `import { exportFile }`
- `await exportText(texts)` → `await exportFile(texts)`
